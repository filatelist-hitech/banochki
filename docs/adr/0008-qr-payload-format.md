# ADR 0008: QR payload v1

QR содержит только `banochki://qr/v1/<opaque-token>`. Token — 32 случайных байта в base64url без padding; в payload нет family id, названия, остатка, места или автора. Парсер сначала проверяет схему и версию; QR v1 остаётся читаемым после появления следующих версий.

Токен стабилен при редактировании партии. Отзыв и перевыпуск меняют состояние записи, а не историю старой наклейки.
