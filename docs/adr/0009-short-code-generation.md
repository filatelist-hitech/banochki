# ADR 0009: короткий номер QR

Формат v1 — `XXXXXX-C`: шесть случайных десятичных цифр и контрольная цифра Luhn. Номер не является SQLite autoincrement. Перед вставкой repository проверяет коллизию, затем SQLite unique index `(family_id, short_code)` остаётся окончательной защитой; при коллизии генерация повторяется. Это сохраняет пригодность для будущего offline multi-device R3.

Номер не переиспользуется: отозванные и заменённые записи остаются в `qr_codes`.
